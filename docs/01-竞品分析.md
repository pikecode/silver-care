# 「家邻康」竞品分析报告

> 分析时间：2026年3月 | 方向：社区居家养老健康管理

---

## 一、市场概览

### 1.1 宏观数据

| 指标 | 数据 |
|------|------|
| 中国 60 岁以上人口 | 超 3.1 亿（2025），占总人口约 22% |
| 银发经济市场规模 | 约 7 万亿元（2024），预计 2035 年达 30 万亿 |
| 养老模式分布 | 9073 格局：居家 90%，社区 7%，机构 3% |
| 智能手机渗透率（60+） | 约 72%（2024），微信使用率超 85% |
| 政策支持 | 2024 年国务院发布《关于发展银发经济增进老年人福祉的意见》 |

### 1.2 关键趋势

1. **居家养老是绝对主流** — 90% 老人不愿离开家，社区服务是核心载体
2. **子女代付意愿强** — 60% 的养老消费由子女发起/支付
3. **从"养老"到"享老"** — 新一代老人（60-70后）消费观念更开放
4. **政策红利** — 政府大力推进社区养老服务设施建设，鼓励社会资本参与
5. **数字化缺口** — 大量社区养老仍停留在纸质登记、人工管理阶段

## 二、竞品矩阵

### 2.1 直接竞品（社区养老 + 数字化）

#### 1）安康通

| 维度 | 详情 |
|------|------|
| 定位 | 智慧养老综合服务平台 |
| 模式 | 政府 B2G + 社区运营 |
| 核心功能 | 紧急呼叫、健康监测、上门服务、日间照料 |
| 覆盖 | 200+ 城市，服务 1000 万+ 老人 |
| 融资 | 已上市（A股，安康通控股前身） |
| 优势 | 政府关系深、覆盖广、品牌认知度高 |
| 劣势 | 产品陈旧，用户体验差，过度依赖政府采购 |
| 启示 | **政府渠道很重要，但不能只靠 B2G** |

#### 2）颐家（上海颐家）

| 维度 | 详情 |
|------|------|
| 定位 | 社区居家养老服务运营商 |
| 模式 | 线下服务站 + 线上平台 |
| 核心功能 | 上门护理、助餐助浴、康复训练、日间托老 |
| 覆盖 | 上海为主，200+ 社区服务点 |
| 融资 | B 轮，累计数亿元 |
| 优势 | 服务标准化程度高，护理人员培训体系成熟 |
| 劣势 | 重资产模式，扩张慢，主要服务高龄失能老人 |
| 启示 | **服务质量是核心壁垒，但不要一开始就做重** |

#### 3）小帮家（智慧社区养老）

| 维度 | 详情 |
|------|------|
| 定位 | 社区养老数字化管理平台 |
| 模式 | SaaS 平台 + 小程序 |
| 核心功能 | 健康档案、服务预约、活动管理、子女端 |
| 覆盖 | 二三线城市社区 |
| 融资 | 天使轮 |
| 优势 | 产品轻，适配社区场景 |
| 劣势 | 用户量小，商业模式未验证 |
| 启示 | **验证了小程序切入社区的可行性，但缺乏差异化** |

## 三、竞争空白与机会

### 3.1 市场空白点

```
                    社区落地能力
                    强 ────────── 弱
                    │              │
          重        │  颐家        │  平安好医生
          │         │  安康通      │
  技术    │         │              │
  轻重    │─────────┼──────────────│
          │         │              │
          轻        │ 【家邻康】    │  腾讯健康
                    │  机会区域    │  微信运动
                    │              │
```

**核心机会：轻技术 + 强社区落地 + 子女付费 的交叉地带**

### 3.2 家邻康的差异化定位

| 差异点 | 竞品现状 | 家邻康策略 |
|--------|----------|------------|
| 子女端体验 | 多数竞品忽略子女端 | **子女是第一付费用户**，体验优先 |
| 获客方式 | 线上投放或政府分配 | **社区驿站线下信任获客** |
| 产品复杂度 | 功能堆砌，老人不会用 | **极简设计，3 步完成核心操作** |
| 商业模式 | 依赖政府采购 | **C端订阅（子女付费）+ 服务佣金** |
| 数据价值 | 未充分利用 | **健康趋势分析，异常预警** |

## 四、总结与建议

### 4.1 竞争策略

1. **不做大而全** — 不跟安康通/平安好医生比资源，做"社区级"精品
2. **子女端为先** — 这是大多数竞品的盲区，也是付费转化的关键
3. **线下信任，线上留存** — 社区驿站建立信任，小程序维持日活
4. **先打一个社区，做出标杆** — 不急于扩张，先做出复购和口碑

### 4.2 进入壁垒评估

| 壁垒类型 | 评估 | 说明 |
|----------|------|------|
| 技术壁垒 | 低 | 小程序技术成熟，开发门槛不高 |
| 运营壁垒 | 中 | 需要社区关系和线下运营能力 |
| 资金壁垒 | 低 | 轻模式启动，10-30 万可以跑通 MVP |
| 品牌壁垒 | 中 | 老年市场信任成本高，需要时间积累 |
| 政策壁垒 | 低 | 政策鼓励方向，无特殊资质要求 |

**综合评估：适合小团队切入，关键在运营执行力而非技术或资金。**
